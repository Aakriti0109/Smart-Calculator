# 🧮 Smart Calculator (C++)

A simple **Smart Calculator** built using C++ that performs multiple useful calculations in one program.
It combines basic arithmetic operations with health and financial utilities.

---

## 🚀 Features

### 1. Basic Calculator

* Perform operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)

---

### 2. BMI Calculator

* Calculates **Body Mass Index (BMI)**
* Takes:

  * Weight (kg)
  * Height (meters)
* Displays:

  * BMI value
  * Category (Underweight, Normal, Overweight, Obese)

---

### 3. Simple Interest Calculator

* Calculates interest using formula:

Simple Interest = (Principal × Rate × Time) / 100

* Inputs:

  * Principal amount
  * Rate of interest (%)
  * Time (years)

---

### 4. Unit Conversion

Supports basic conversions:

* Kilometers → Meters
* Meters → Kilometers
* Celsius → Fahrenheit
* Fahrenheit → Celsius

---

## 🛠️ Technologies Used

* C++
* Standard Library (`iostream`, `cmath`)

---

## ▶️ How to Run

### Step 1: Compile

```bash
g++ smart_calculator.cpp -o calculator
```

### Step 2: Run

```bash
./calculator
```
